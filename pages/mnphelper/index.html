MNP Helper

Определяет оператора телефонной сети по номеру телефона, согласно информации из базы данных переносимых номеров (БДПН), при исходящих и входящих вызовах и показывает эту информацию во всплывающих сообщениях.

Обратите внимание, что при определении оператора приложение запрашивает данные через сеть Интернет из онлайн-базы ЦНИИС (Центральный научно-исследовательский институт связи).

При отсутствии на устройстве Wi-Fi соединения с Интернет, запрос к базе будет проходить с использованием мобильной сети (2G/3G/LTE).

Ваш оператор мобильной сети в режиме 2G может не поддерживать одновременное совершение телефонного вызова и использование сети Интернет, поэтому для работы приложения во время телефонных вызовов возможно потребуется переключение устройства в режим 3G (или 4G/LTE).
